Easy & Simple

기존의 복잡한 작업환경에서 벗어나 쉽고 단순한 인터페이스를 제공합니다.

  • Interface 쉽고 단순한
    인터페이스     사용자의 편의성을 최우선으로 고려하여, 기존의 복잡하고 혼란스러운 인터페이스 대신 간결하고 직관적인 메뉴 구성을 통해 누구나 쉽게 사용 할 수 있는 사용자 친화적인 환경을 제공합니다.
  • Grouping 기능별로 그룹화하여
    차별화된 디자인     복잡한 인터페이스를 간소화하고, 기능별로 메뉴를 체계적으로 그룹화 하여 사용자가 필요한 기능을 쉽게 찾을 수 있도록 구성했습니다.
  • Imaging 정보전달 방식의
    이미지화     객체의 속성 정보를 사용자기 쉽게 이해할 수 있도로 이미지와 함께 시각적으로 표현하여 정보전달의 명확성를 높였습니다
    • 객체 종류에 따른 차별화된 속성정보 창 제공
    • 객체의 기본정보, 형상정보, 위치정보 구분하여 제공
    • 많은 정보의 가독성 및 가시성 증대로 작업성 향상
